Welcome to the Cube Escape Collection

Cube Escape is the first game series created by Rusty Lake, introducing the strange and mysterious universe that defines their games.

This collection includes the first nine Cube Escape chapters, originally released as Flash games. They have been preserved and updated to ensure they remain playable after the end of Flash support.

This version is a remake and browser port developed by Cosmos, bringing the Cube Escape experience to modern web browsers while staying faithful to the original games created by Rusty Lake.
